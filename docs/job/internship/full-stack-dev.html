全栈开发实习

从前后端编写到云部署的全流程，使用JavaScript、Python、Java，以及MySQL和PostgreSQL数据库，依托AWS和Azure平台，提供全面的现代应用开发经验

项目时长

10周 / 40小时

适合学员

求职SDE方向且缺少美国实习经验的留学生

项目特色

  • 进行真实移动端软件开发
  • 提升前后端-编程能力
  • 增加在美实习经验

实习介绍

实习生在一家科技公司任职，参与开发一个与小红书相似的食物分享应用的移动端项目。此职位涵盖了全面的项目开发流程，包括前后端代码的编写、优化和最终部署到云平台。您将负责设计数据结构，参与DevOps相关工作，确保项目从概念到发布的每个阶段都高效、顺利进行。在技术选型方面，项目将采用JavaScript、Python和Java作为主要开发语言，使用MySQL和PostgreSQL作为数据库解决方案，同时，云基础设施将主要依托于AWS和Azure平台。此岗位旨在为实习生提供一个全面深入了解和参与现代应用开发的机会。

项目大纲

  • 第一部分
App后端开发
上课频率:5周
4小时课程/周
这部分用Node.js, Express.js, MongoDB等技术进行App后端开发。

  • 第1周：项目介绍与后端环境搭建

    • 介绍导师背景和项目概览
    • 学习Express.js、MongoDB基础和设置开发环境
    • 进行后端简历会话

  • 第2周：后端架构设计与API开发

    • 设计后端架构，学习如何使用RESTful API
    • 设置AWS账户

  • 第3周：用户管理与认证系统

    • 实现认证与验证功能，开发用户相关的APIs
    • 提供后端Office Hour以解答疑问

  • 第4周：视频服务与云基础

    • 开发视频相关的APIs，学习AWS基础知识
    • 继续提供后端Office Hour

  • 第5周：部署准备与系统监控

    • 准备部署、设置CI/CD流程，学习使用DataDog进行数据监控
    • 进行后端回顾与改进
